## Even G2 Development Notes

独立してメンテナンスされている包括的な開発ノート。

- リポジトリ: https://github.com/nickustinov/even-g2-notes/blob/main/docs/README.md
- 内容: アーキテクチャ詳解、Unicode グリフの完全な対応表、SDK の癖、エラーコード、注釈付きサンプル
- 実装例: チェス、Reddit、天気、Tesla 車両ステータス、Pong、Snake など(各種アーキテクチャパターンの実例)

## Even Toolkit

Web UI(設定ページ・構成画面)向けのコンポーネントライブラリ。

- リポジトリ: https://github.com/fabioglimb/even-toolkit
- インストール: `npm install even-toolkit`

| 含まれるもの | 詳細 |
|---|---|
| React コンポーネント 55+ | Button / Card / NavBar / ListItem / Toggle / Dialog / Toast / BottomSheet / Charts / Calendar など |
| ピクセルアートアイコン 191 個 | グラス向け |
| ユーティリティ | `useGlasses` フック、`buildActionBar`、`mapGlassEvent` |
| デザイントークン | CSS カスタムプロパティ(色・間隔・フォント FK Grotesk Neue)、タイポグラフィ(`.text-vlarge-title` 24px 〜 `.text-detail` 11px) |
