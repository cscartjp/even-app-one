# ディスプレイと UI システム(Display & UI System)

> 原文: https://hub.evenrealities.com/docs/guides/display
> ※ 本ページは公式ドキュメントの非公式な日本語要約です。

## ディスプレイ仕様

| 項目 | 仕様 |
|---|---|
| キャンバス | 片眼あたり **576 × 288 ピクセル** |
| 座標系 | 原点は左上。X は右方向、Y は下方向 |
| 色 | **4bit 階調(16 レベル)の緑色**のみ |

## UI システムの考え方

**CSS・flexbox・DOM は存在しない。** 代わりに「コンテナ」を配置する方式。

### コンテナの制限

- 画像コンテナ: 最大 **4 つ**
- その他のコンテナ(テキスト・リスト): 最大 **8 つ**
- **必ず 1 つのコンテナに `isEventCapture: 1` を設定**する
- コンテナは重ねられるが、重なり順は**宣言順のみ**で制御(Z-index なし)

### 全コンテナ共通プロパティ

| プロパティ | 型 | 範囲 |
|---|---|---|
| `xPosition` | number | 0–576 |
| `yPosition` | number | 0–288 |
| `width` | number | 0–576 |
| `height` | number | 0–288 |
| `containerID` | number | — |
| `containerName` | string | 最大 16 文字 |
| `isEventCapture` | number | 0 または 1 |

## テキストコンテナ

- 平文テキストを**左寄せ・上寄せ**で描画
- テキスト配置・フォントサイズ・太字/斜体の制御は**できない**
- `\n` で改行。Unicode 文字をサポート
- 全画面でおよそ **400〜500 文字**が目安
- `textContainerUpgrade` でちらつきなしの高速更新が可能

| 操作 | 最大文字数 |
|---|---|
| `createStartUpPageContainer` | 1,000 文字 |
| `textContainerUpgrade` | 2,000 文字 |

## リストコンテナ

- 最大 **20 アイテム**、各アイテム最大 **64 文字**
- スクロールとフォーカスのハイライトは**ファームウェアが処理**してくれる
- カスタムスタイル・アイテム高さ・区切り線は不可
- **インプレース更新不可**(変更にはページ全体のリビルドが必要)

## 画像コンテナ

- サイズ: 幅 **20–200px**、高さ **20–100px**
- 4bit 階調(グレースケール)
- データ形式: `number[]` / `Uint8Array` / `ArrayBuffer` / base64
- `createStartUpPageContainer` 時には画像データを送れない(生成後に `updateImageRawData` で送る)
- **並行送信は不可**(直列化必須)

## ボーダー(テキスト・リストコンテナのみ)

| プロパティ | 範囲 |
|---|---|
| `borderWidth` | 0–5 |
| `borderColor` | 0–15(階調レベル) |
| `borderRadius` | 0–10 |
| `paddingLength` | 0–32 |

> ⚠️ 背景色・塗りつぶしのプロパティは存在しない。**唯一の視覚装飾はボーダー**。

## フォントと Unicode

- 単一の LVGL フォントのみ。サイズ変更不可、モノスペースではない
- サポート外の文字は**自動的にスキップ**される

### UI 構築に便利な文字

| 用途 | 文字 |
|---|---|
| プログレスバー | `━` `█▇▆▅▄▃▂▁` |
| ナビゲーション | `▲△▶▼▽◀` |
| 選択状態 | `●○` `■□` |
| 枠線 | `╭╮╯╰` `│─` |
